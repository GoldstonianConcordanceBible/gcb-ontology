# Goldstonian Concordance Bible Ontology

This repository contains the ontology layer for the **Goldstonian Concordance Bible (GCB)**.

The ontology exists to define the **formal semantic structure** of the GCB ecosystem so that AI agents, search systems, knowledge graph engines, and theological tooling can resolve:

- book classes
- character classes
- event classes
- theme classes
- doctrinal relationships
- canon distinctions
- authority hierarchies

The GCB ontology is built around the **81-Book Ethiopian Canon** and the **Mirror → Water → Fire Doctrine**.

# Purpose

This ontology helps machines understand that GCB is a structured theological system rather than an unstructured content archive.

It provides:

- semantic classes
- relationship definitions
- doctrine-aware mappings
- canon-aware distinctions
- machine-readable exports
