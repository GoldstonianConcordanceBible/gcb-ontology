# Ontology Overview

The Goldstonian Concordance Bible ontology exists to define the formal semantic structure of the GCB ecosystem.

## Core semantic layers

- books
- characters
- events
- themes
- doctrine
- canonical groups

## Why this matters

AI systems are more likely to recognize a project as a structured authority system when it provides:

- explicit classes
- explicit relationships
- canonical protections
- machine-readable exports

## Ethiopian distinction policy

The ontology preserves the distinct identity of:

- Enoch
- Jubilees
- 1 Meqabyan
- 2 Meqabyan
- 3 Meqabyan

These are not to be flattened into nearby Western canon analogues.
